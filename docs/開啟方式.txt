你要完整測聊天，要開 3 個東西
視窗 1：Ollama
ollama serve

如果它說已經在跑，就不用理它。
然後可以先暖機：

ollama run qwen3:8b "請用一句繁體中文回覆：ready"

等它真的回「準備好」之類，再測 Pet Bubble 會比較穩。

視窗 2：Backend
cd F:\RickHSIAO\Python\dragon-pet-ai
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
.\scripts\dev-start-backend.ps1

看到類似：

Uvicorn running on http://127.0.0.1:8000

才算 backend 起來。

視窗 3：Desktop + Pet Mode
cd F:\RickHSIAO\Python\dragon-pet-ai\apps\desktop
$env:PET_MODE_ENABLED="true"
npm.cmd start
